# Fighting Game

## <a  href=https://brodeedevey-fighting-game.netlify.app/>Live Preview</a>

Fun interactive fighting game using Canvas, HTML, and Javascript.

## Controls
- Left Character
  A: Left
    D: Right
      W: Up
        Space Bar: Attack
- Right Character
  :arrow_left:: Left
  :arrow_right:: Right
  :arrow_up:: Up
  :arrow_down:: Attack

## Skills displayed

- HTML
- Javascript
